If its your first startup and you would like to use the file functions. Please run main.py and then back out at the main menu.
It will create all the necesarry folders that this service needs.

Please place all photos of faces in the /data/faces folder.
Please place all .wav files in the /data/audio folder.

When prompted to input the name, enter the name of the file including the extension. It will automatically find the file,
if you have placed in the correct folder. 

All file outputs will be put in the /output folder.
